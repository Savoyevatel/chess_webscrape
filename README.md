# Chess Openings App

A web application for browsing and saving chess openings. Visit https://chessfox.com/chess-openings-list/ for more in depth learning and information!.

## Features

- View chess openings and their move sequences.
- Save favorite openings for later reference.
- Navigate between different openings.
- No user authentication required.

## Technologies Used

- Frontend: React, React Router, Bootstrap
- Backend: Django, Django REST Framework
- Chess logic: Chess.js
- Web scraping: BeautifulSoup, requests
